# Dr. Kathuria's Dentistry — Website

Premium dental clinic website for **Dr. Kathuria's Dentistry**, East of Kailash, New Delhi. NABH Accredited, ISO Certified, trusted by 15,000+ patients from 44+ countries.

## Live Preview

[https://dr-kathuria-dentistry.netlify.app](https://dr-kathuria-dentistry.netlify.app)

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations, glassmorphism, responsive design
- **Vanilla JavaScript** — No frameworks, no dependencies
- **Font Awesome 6.5** — Icons
- **Google Fonts** — Cormorant Garamond + DM Sans
- **Netlify Forms** — Contact form submissions

## Project Structure

```
dkd/
├── index.html          # Homepage (all sections)
├── css/
│   └── style.css       # Complete design system + responsive breakpoints
├── js/
│   └── main.js         # Interactive features & animations
└── README.md
```

## Features

### Sections
- Hero slider (3 slides with auto-rotation)
- Info bar (Google rating, phone, clinic hours)
- Scrolling ticker + certification marquee
- Animated stats counter (15K+ patients, 44+ countries, 18+ years, 4.8 rating)
- About preview with credentials
- 6 Service cards with pricing (Veneers, Whitening, Implants, Aligners, Root Canal, Gum Treatment)
- Why Choose Us (5 reasons)
- Before & After drag slider
- How It Works (4-step process)
- Testimonials carousel (4 international patients)
- Celebrity smiles grid (6 cards)
- Video consultation / Teledentistry section
- Advanced technology showcase (4 cards)
- Google reviews (3 verified reviews)
- 44+ Countries reach with flag pills
- Clinic gallery with lightbox (8 images)
- Instagram feed grid (6 posts)
- Smile Confidence Score quiz (5 questions, instant results)
- CTA banner
- Full footer with contact info & social links

### Interactive Components
- **Hero Slider** — Auto-play, arrows, dot navigation
- **Testimonial Carousel** — Auto-rotate with pip navigation
- **Before/After Slider** — Drag to compare results
- **Smile Quiz** — 5-question quiz with score ring animation & treatment recommendations
- **Lightbox** — Click-to-zoom clinic gallery
- **Floating Appointment Panel** — Slide-in booking form
- **Animated Counters** — Number animation on scroll into view
- **Scroll Reveal** — Fade-in animations on scroll
- **Sticky Navigation** — Shrinks on scroll with shadow
- **Mobile Navigation** — Hamburger menu with slide-in panel
- **Side Contact Strip** — Expandable social links (desktop)
- **FAB Buttons** — Floating call & WhatsApp buttons (mobile)
- **Scroll to Top** — Appears after 400px scroll
- **Toast Notifications** — Form submission feedback
